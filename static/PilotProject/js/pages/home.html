PrepAir (Pretty clever, huh?)

PrepAir is built on Angular/Django, as a Single-Page-Application. It gathers up-to-date airport information from an external API at flightplandatabase.com. It also stores other relevant information in an internal Django database. This internal database is so that users can log additional information on an airport (likes, comments, pilot reports, etc.) that can be stored and accessed alongside dynamic, real-time airport information.
