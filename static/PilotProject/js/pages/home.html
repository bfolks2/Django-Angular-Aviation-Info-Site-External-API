PrepAir (Alpha Build)

(Pretty clever, huh?)

PrepAir is built using Angular on the Front-End and Python/Django on the Back-End. It is contructed as a Single-Page-Application. It gathers up-to-date airport information from an external API at flightplandatabase.com. It also stores other relevant information in an internal Django database. This internal database allows user to log additional information on an airport (likes, comments, pilot reports, etc.) that can be stored and accessed alongside dynamic, real-time airport information.

The airport data on this site is compiled to aid in creating a flight plan simulation.
